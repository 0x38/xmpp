Library for XMPP protocol connections (Jabber) for PHP.

## SYSTEM REQUIREMENTS

- PHP 5.3
- clue/socket-raw - a wrapper class for PHPs socket functions
- psr/log - for PSR-3 logging

## INSTALLATION

Add this to your composer file:

    {
        "require": {
            "fabiang/xmpp": "*"
        }
    }
