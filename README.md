# README

fabiang/xmpp a XMPP (Jabber) client library for PHP 5.3

# SYSTEM REQUIREMENTS

- PHP 5.3
- clue/socket-raw - a wrapper class for PHPs socket functions
- psr/log - for PSR-3 logging

# INSTALLATION

Add this to your composer file:

    {
        "require": {
            "fabiang/xmpp": "*"
        }
    }
